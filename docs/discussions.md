# Discussions

Use Discussions for general Q&A, ideas, announcements, and conversations that are not yet a tracked commitment.

## Recommended categories

- **Announcements** — official notices from maintainers
- **Q&A** — questions that need a clear answer
- **Ideas** — suggestions that need exploration

## Start a Discussion

1. Select **Discussions** and then **New discussion**.
2. Choose the correct category.
3. Use the question or decision as the title.
4. Add context and mention relevant people.
5. Mark the accepted answer when a Q&A is resolved.

When a conversation becomes actionable, create an Issue and link the two.
