# Projects

Use the TB Project board to see work visually without opening every Issue.

## Status guide

| Status | Meaning |
|---|---|
| New | Submitted and awaiting review |
| Triaged | Reviewed, prioritised, and assigned |
| In progress | Actively being worked on |
| Blocked | Waiting on a dependency or decision |
| Done | Delivered and verified |

## Management fields

Each tracked item should have:

- Status
- Priority
- Owner
- Target date

The Issue remains the detailed record. The Project board provides the management view.
