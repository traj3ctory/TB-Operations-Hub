# Issues

Use an Issue when something needs action, investigation, a decision, or an accountable owner.

## Choose a template

### Requirement / Update request

Use this for a new need, requested change, or improvement.

### Problem report

Use this when something is not working or a process has failed.

## What to include

- What is needed or what happened
- Why it matters to TB
- The desired outcome
- Priority and target date
- Supporting evidence or links

Keep one outcome per Issue. Use comments for progress updates and decisions.
