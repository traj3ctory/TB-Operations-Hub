# TB Operations Hub

A simple GitHub workspace for TB programme Management, Executive, M&E and coordination users.

## Start here

- **Docs:** approved procedures, indicator definitions, templates and guidance
- **Issues:** requirements, updates and problems that need action
- **Discussions:** questions, announcements and ideas
- **Project board:** status, priority, owner and target date

The published guide is available at <https://traj3ctory.github.io/TB-Operations-Hub/>.

## Working rule

**Discuss → record → track → publish.** Start in Discussions when the answer is unclear. Open an Issue when action is required. Track it on the Project board. Publish the approved outcome in Docs.

No coding, Git commands or local setup is required for normal users.
